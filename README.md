# Balloon Popping Game

This game is made purely in Intel x86 architecture 8088 assembly language.

## Gameplay and Mechanics

"GUBARAY" is an interactive balloon-popping game with two different game modes:

- **Survival Mode:** In this mode, you have a 2-minute timer and 3 lives. Balloons will spawn from the bottom of the screen with a specific letter on them. You must type the correct letter before the balloon escapes the top of the screen.
- **Endless Mode:** A casual mode with no lives or timer limit. Test your typing speed and try to get the highest score possible!

### Rules & Scoring

- Hitting the correct key for a balloon gives you **+10 points**.
- The game can be paused at any time by pressing the `ESC` key, which opens the pause menu allowing you to resume or quit.
- When the game ends, your final score is displayed on a dedicated Game Over screen.

## Technical Implementation Details

The game is completely written in Intel 8088 Assembly (16-bit DOS architecture) using raw memory-mapped I/O and hardware interrupts:

- **Video Memory Rendering:** Graphics are rendered by directly writing to the CGA/VGA video text segment at memory address `0xB800`. The game uses various ASCII block characters (like `0xDB`) and attributes to draw the balloons, background colors, and the large "GUBARAY" title logo.
- **Hardware Interrupts:**
  - Keyboard input is handled using BIOS interrupt `INT 16h`.
  - The game hooks into the system timer interrupt `INT 08h` (PIT) to create a custom countdown timer and control spawn delays reliably regardless of CPU speed.
  - Basic sound effects (like a "pop" sound and melody) are generated by programming the Programmable Interval Timer (PIT) at port `43h` and sending frequencies to the PC Speaker via port `61h`.
- **Game Loop:** A custom game loop manages the state of 4 active balloons concurrently, recalculating coordinates and checking against the player's keystrokes.
